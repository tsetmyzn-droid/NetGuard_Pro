Currently supports [OpenWrt](https://openwrt.org/) version 19.07 or 21.02 or 22.03 or 23.05 or 24.10 with [LuCI](https://openwrt.org/packages/pkgdata/luci) installed.

## Development Setup
Install the Flutter SDK for your system.

Use `flutter doctor` to see steps required for your system.

## Debugging
See available emulators
`flutter emulators`

Launch emulator
`flutter emulators --launch your_emulator_id`

Run program on the emulator
`flutter run`
