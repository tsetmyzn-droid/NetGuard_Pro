# 🧠 برومت تقني: تطوير إضافة OpenWrt لجلب استهلاك البيانات (NetGuard Pro Extension)

## 📌 الهدف
بناء نظام متكامل لإضافة (Agent) تُثبَّت على راوتر OpenWrt لجمع بيانات استهلاك كل جهاز متصل وحالة الاتصال.

## 🎯 الميزات المطلوبة
1. **جمع الاستهلاك:** (Download/Upload تراكمي) وحالة الاتصال.
2. **التثبيت التلقائي:** من التطبيق عند اكتشاف غياب الإضافة.
3. **بروتوكول آمن:** تواصل مشفر وبمصادقة.
4. **العزل الأمني:** إيقاف ذاتي عند حدوث أخطاء جسيمة.
5. **سجلات الأخطاء:** إرسال السجلات للتطبيق.

## 📦 المكونات التقنية
- **الوكيل (Agent):** Python CGI script يقدم البيانات بصيغة JSON.
- **تجميع البيانات:** الاعتماد على `nlbwmon`.
- **المصادقة:** X-API-Key مع HMAC-SHA256.
- **المسار:** `/cgi-bin/traffic`.

*ملاحظة: هذا المستند مرجع لتطوير الميزة في مراحل الاستقرار القادمة.*
