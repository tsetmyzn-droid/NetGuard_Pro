# 🛡️ AGENT_SECURITY_ARCHITECTURE.md

تصميم معماري للأمان في NetGuard Agent يتبع مبادئ Zero-Trust و Defense-in-Depth.

## 🔐 نظام المصادقة والنزاهة (Authentication & Integrity)

### 1. HMAC-SHA256 Signing
كل طلب من التطبيق للراوتر يجب أن يحتوي على توقيع HMAC:
- **المكونات**: `Signature = HMAC(SharedKey, Method + Path + Timestamp + Nonce + Body)`.
- **الهدف**: منع تعديل الطلبات أثناء الإرسال والتأكد من هوية التطبيق.

### 2. الحماية من إعادة الإرسال (Replay Protection)
- **Timestamp**: الطلبات التي يتجاوز عمرها 30 ثانية يتم رفضها.
- **Nonce**: (رقم يستخدم مرة واحدة) يتم تخزين آخر 20 Nonce في ذاكرة الراوتر المؤقتة لمنع تكرار نفس الطلب.

---

## 🔒 حماية السطح (Attack Surface Minimization)

- **LAN Only**: الوكيل يستمع فقط على واجهة `br-lan`. لا يمكن الوصول إليه عبر الـ WAN نهائياً.
- **Minimal API**: لا توجد وظائف لتنفيذ أوامر Shell عشوائية (`No exec()`). الوكيل يدعم فقط مسارات محددة (Read Stats, Write Config).
- **Least Privilege**: يعمل سكريبت الوكيل بصلاحيات مستخدم محدود (مثل `nobody`) حيثما أمكن، مع الوصول للملفات الضرورية فقط.

---

## 🗝️ إدارة المفاتيح (Key Management)

- **Key Generation**: يتم توليد المفتاح المشترك (Shared Key) عشوائياً عند أول تثبيت للوكيل ويرسل للتطبيق عبر SSH (محمي).
- **Secure Storage**: يخزن المفتاح في التطبيق باستخدام الـ `Biometric / KeyStore` وفي الراوتر في ملف بصلاحيات `600` (لـ root فقط).

---

## 🛡️ آليات الدفاع النشط (Active Defense)

- **Rate Limiting**: بحد أقصى 5 طلبات في الثانية من نفس الـ IP.
- **Brute Force Detection**: إذا فشلت المصادقة 5 مرات متتالية، يتم حظر الـ IP الطالب لمدة 10 دقائق وكتابة سجل أمان.
- **Tamper Detection**: يقوم الوكيل بفحص بصمة ملفاته الأساسية عند كل تشغيل للتأكد من عدم تعديلها.

---

## ⚖️ مقايضات الأمان (Security Trade-offs)

- **HTTPS vs HMAC**: في الراوترات الضعيفة، عبء تشفير TLS (HTTPS) قد يكون ثقيلاً. لذا نفضل **HMAC-over-HTTP** كخيار افتراضي لأنه يوفر النزاهة والمصادقة بجهد معالج أقل بكثير، مع ترك خيار تفعيل HTTPS للراوترات القوية.

---
**القاعدة الذهبية**: "إذا شك الوكيل، فإنه يتوقف". أي سلوك مشبوه أو فشل متكرر في المصادقة سيؤدي لإغلاق الوكيل فوراً (Self-Disable).
