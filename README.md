# NetGuard Pro V7 🛡️

**NetGuard Pro** هو تطبيق متقدم وشامل لإدارة الشبكات وأمن أجهزة التوجيه (Routers). تم تصميمه ليوفر للمستخدم العادي والمحترف واجهة عصرية وسهلة للتحكم الكامل في بيئة الشبكة المنزلية أو المكتبية، مع دمج تقنيات الذكاء الاصطناعي لتعزيز الأداء والأمان.

---

## 🏗️ البنية التقنية (Technical Stack)

تم بناء التطبيق باستخدام أحدث التقنيات لضمان السرعة، الأمان، والتوافق العابر للمنصات:

- **Frontend**: React 19 + TypeScript + Vite.
- **Styling**: Tailwind CSS 4.0 (للتصميم العصري والاستجابة الكاملة).
- **Animations**: Motion (Framer Motion) لتجربة مستخدم سلسة وتفاعلية.
- **Cross-Platform**:
  - **Windows**: Electron (لتحويل التطبيق إلى نسخة سطح مكتب EXE).
  - **Android**: Capacitor (لتحويل التطبيق إلى نسخة جوال APK).
- **AI Integration**: Google Gemini API (للترجمة الذكية وتحليل استهلاك الشبكة).
- **CI/CD**: GitHub Actions (للبناء الآلي وتوليد النسخ النهائية).

## 🚀 الميزات الاحترافية

### 1. لوحة تحكم ذكية (Smart Dashboard)
- **مراقبة لحظية**: رسوم بيانية تفاعلية تظهر سرعة التحميل والرفع وتحديثات كل 3 ثوانٍ.
- **إحصائيات النظام**: مراقبة استهلاك المعالج (CPU) والذاكرة (RAM) للراوتر.
- **إجراءات سريعة**: إعادة تشغيل الراوتر، تفعيل واي فاي الضيوف، وفحص الأمان بضغطة زر.
- **الوضع الليلي (Dark Mode)**: دعم كامل للوضع الداكن المريح للعين مع إمكانية التبديل التلقائي.

### 2. إدارة الأجهزة المتقدمة (Device Management)
- **قائمة الأجهزة**: عرض تفصيلي لكل جهاز (IP, MAC, النوع، الحالة).
- **التحكم بالوصول**: إمكانية حظر أو إلغاء حظر أي جهاز فوراً.
- **تتبع الاستهلاك التفصيلي**: 
  - معرفة كمية البيانات لكل جهاز.
  - **استهلاك التطبيقات**: تتبع استهلاك كل تطبيق بالاسم (YouTube, Netflix, etc).
  - **نوع المحتوى**: تصنيف الاستهلاك (فيديو، ألعاب، تواصل اجتماعي).

### 3. تحليلات معمقة (Advanced Analytics)
- **سجل الأحداث**: تتبع دخول وخروج الأجهزة ومحاولات الاختراق.
- **توزيع الاستهلاك**: رسوم بيانية دائرية تظهر توزيع الاستهلاك حسب نوع المحتوى.
- **الجدول الزمني المرن**: عرض استهلاك البيانات على مدار **اليوم، الأسبوع، أو الشهر** بدقة عالية.

### 4. الاتصال والبروتوكولات (Connectivity)
- **دعم متعدد البروتوكولات**:
  - **SSH (Paramiko)**: للتحكم المباشر عبر الأوامر.
  - **API (Requests)**: للاتصال السريع والآمن عبر واجهات البرمجة.
  - **WEB (Selenium)**: لاستخراج البيانات من لوحات التحكم التقليدية.
- **عنوان الراوتر**: إمكانية تحديد عنوان الوصول (IP) يدوياً لضمان التوافق مع كافة الشبكات.

### 5. محسن الواي فاي (Wi-Fi Optimizer)
- **تحليل القنوات**: فحص القنوات المحيطة واقتراح القناة الأفضل لتقليل التداخل.
- **درع الأمان**: فحص شامل للثغرات الأمنية في إعدادات الراوتر.

### 5. الذكاء الاصطناعي (AI Core)
- **ترجمة فورية**: دعم كامل للغتين العربية والإنجليزية مع واجهة RTL.
- **تحليل ذكي**: يقوم الذكاء الاصطناعي بتحليل أنماط الاستهلاك وتقديم نصائح لتحسين الأداء.

## 📱 التحميل والتثبيت (GitHub Actions)

يتم بناء التطبيق تلقائياً عند كل تحديث للكود. للحصول على النسخ الجاهزة:

1. اذهب إلى صفحة **Actions** في مستودعك.
2. اختر آخر عملية بناء (Workflow Run).
3. حمل الملفات من قسم **Artifacts**:
   - `NetGuard-Pro-Android`: ملف APK للجوال.
   - `NetGuard-Pro-Windows`: ملف EXE للكمبيوتر.

## 🛠️ دليل المطورين (Manual Build)

### المتطلبات الأساسية:
- **Node.js**: إصدار 24 أو أحدث (مهم جداً للتوافق مع Capacitor 8).
- **Java**: JDK 17 (لبناء الأندرويد).
- **Android Studio**: لإدارة بيئة الأندرويد.

### خطوات التشغيل:
```bash
# 1. تثبيت المكتبات
npm install

# 2. تشغيل نسخة الويب للتطوير
npm run dev

# 3. بناء نسخة الويندوز
npm run electron:build

# 4. بناء نسخة الأندرويد
npm run build
npx cap sync
# ثم استخدم Android Studio لفتح مجلد android وتوليد APK
```

## 🛡️ الأمان والخصوصية
NetGuard Pro يتبع نهج **Privacy-First**:
- لا يتم إرسال أي بيانات خاصة إلى سيرفرات خارجية.
- التشفير يتم محلياً باستخدام AES-256.
- الاتصال بالراوتر يتم عبر بروتوكولات آمنة (SSH/HTTPS).
