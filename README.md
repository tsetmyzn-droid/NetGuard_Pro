# NetGuard Pro V7 🛡️

**NetGuard Pro** هو تطبيق متقدم لإدارة الشبكة وأمن الراوتر، يجمع بين ميزات التحكم الكامل، المراقبة اللحظية، والحماية الأمنية القصوى في واجهة واحدة عصرية.

## 🚀 الميزات الرئيسية

### 1. التحكم الكامل بالراوتر (Router Chef Mode)
- **إدارة الأجهزة**: عرض جميع الأجهزة المتصلة مع إمكانية حظر أي جهاز غير مرغوب فيه.
- **تحديد السرعة (Bandwidth Control)**: تحديد حد أقصى للسرعة لكل جهاز لضمان توزيع عادل للإنترنت.
- **الجدولة الزمنية**: تحديد أوقات محددة لعمل الإنترنت لكل جهاز (مثالي للرقابة الأبوية).
- **تحسين الواي فاي (Wi-Fi Optimizer)**: فحص القنوات واختيار القناة الأقل تداخلاً لزيادة استقرار الإشارة.

### 2. المراقبة اللحظية (Network Monitor 1.0)
- **تحديثات كل 500ms**: متابعة استهلاك البيانات والسرعة بدقة متناهية في الوقت الفعلي.
- **نظام الكوتا (Quota System)**: تحديد حد أقصى للاستهلاك الشهري مع تنبيهات بصرية عند الاقتراب من الحد.
- **سجل الجلسات**: تتبع مدة كل جلسة اتصال وكمية البيانات المستهلكة فيها.

### 3. الأمن والخصوصية (Security Shield)
- **تشفير AES-256**: يتم تشفير جميع كلمات مرور الراوتر والبيانات الحساسة محلياً.
- **كشف Hydra**: نظام ذكي لكشف ومنع هجمات القوة الغاشمة (Brute Force).
- **حماية MITM**: التحقق من هوية الراوتر لمنع هجمات "الرجل في المنتصف".
- **درع الخصوصية (Privacy Shield)**: منع لقطات الشاشة، منع النسخ، ومنع النقر الأيمن لحماية بياناتك.

### 4. مركز المساعدة (Help Center)
- **شرح الميزات**: دليل تفاعلي يشرح كيفية استخدام كل ميزة في التطبيق (Dashboard, Optimizer, Security, Devices).
- **سهولة الوصول**: متاح دائماً من القائمة الجانبية أو الشريط السفلي.

### 5. العمل بدون إنترنت (Local-First)
- التطبيق يتصل بالراوتر عبر الشبكة المحلية مباشرة، مما يعني أنه يعمل بكفاءة حتى لو كانت خدمة الإنترنت مقطوعة.

## 📱 التحميل والتثبيت

يتم بناء التطبيق تلقائياً لكل من الأندرويد والويندوز عبر GitHub Actions.

### للحصول على التطبيق:
1. اذهب إلى تبويب **Actions** في مستودع GitHub الخاص بك.
2. اختر آخر عملية بناء ناجحة (Build NetGuard Pro).
3. في أسفل الصفحة، ستجد قسم **Artifacts** يحتوي على:
   - `NetGuard-Pro-Android`: ملف APK للتثبيت على الجوال.
   - `NetGuard-Pro-Windows`: ملف EXE للتشغيل على الكمبيوتر.

## 🛠️ للمطورين (البناء اليدوي)

إذا كنت ترغب في بناء التطبيق يدوياً على جهازك:

### المتطلبات:
- Node.js >= 22
- npm

### الأوامر:
```bash
# تثبيت المكتبات
npm install

# تشغيل نسخة الويب (للمعاينة)
npm run dev

# بناء نسخة الويندوز (EXE)
npm run electron:build

# بناء نسخة الأندرويد (APK)
npm run cap:sync
# ثم افتح مجلد android في Android Studio لبناء الـ APK
```
