# 🛡️ NetGuard Pro (Native Engine)

**Version:** 5.0.0 (Production Ready)  
**Core:** Flutter (Dart) Native  
**Architecture:** Plugin-Based Local Management

---

## 🚀 Overview
**NetGuard Pro** is a powerful, privacy-focused network management tool designed specifically for power users. Unlike other solutions, NetGuard Pro operates **100% locally**, ensuring your router credentials and network data never leave your device. It provides a "GlassWire-style" visual experience for your home or office router.

## ✨ Key Features
- **Direct Router Link:** Connects directly to your gateway (Huawei, ZTE, TP-Link, OpenWrt).
- **Real-Time Traffic Monitoring:** High-fidelity graphs showing download and upload speeds.
- **Node Management:** View and monitor all connected devices on your network.
- **Plugin System:** Extensible architecture supporting multiple router brands via LuCI RPC and other protocols.
- **Privacy by Design:** No cloud, no account required, no internet access needed for the app itself (Local Only).

## 📱 Supported Platforms
- **Android:** Fully optimized native performance.
- **Desktop (Windows/Linux/macOS):** Consistent experience across all major operating systems.

## 🛠️ How to Run
1. Ensure you have the [Flutter SDK](https://flutter.dev) installed.
2. Clone the repository.
3. Run `flutter pub get`.
4. Connect your device or use an emulator.
5. Launch with `flutter run`.

## 🧩 Plugin Ecosystem
NetGuard Pro currently supports:
- **OpenWrt (LuCI RPC):** Full integration for advanced network control.
- **Huawei/ZTE/TP-Link:** Managed via native protocol implementations.
