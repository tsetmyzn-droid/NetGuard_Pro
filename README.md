# 🛡️ تقرير التدقيق التقني: NetGuard Pro

هذا المستند يمثل الحالة الراهنة للمشروع بناءً على الكود المصدري الفعلي، بعيداً عن أي توصيفات تسويقية.

| المنصة | الحالة | الملاحظات التقنية |
| :--- | :--- | :--- |
| **Windows (EXE)** | ✅ جاهز | تم التحقق من استقرار الملف التنفيذي. |
| **Android (APK)** | ❌ فاشل | خطأ في إعدادات SDK داخل Workflow البناء (قيد الإصلاح). |

---

## 🛠️ الهيكل التقني (Technical Stack)

*   **اللغة الأساسية**: Python 3.10+
*   **إطار عمل الواجهة**: Flet (Flutter for Python).
*   **قاعدة البيانات**: SQLite (مع تشفير Fernet للبيانات الحساسة).
*   **المحرك**: محرك موحد (Monolithic) يجمع بين المنطق والواجهة في ملف واحد.

## ⚙️ الميزات المكتملة (Implemented Features)

هذه الميزات تم كتابة كودها الفعلي وهي تعمل في البيئة الحالية:
1.  **مراقبة الواجهات الشبكية**: جلب بيانات الاستهلاك اللحظي من نظام التشغيل باستخدام `psutil`.
2.  **ماسح الشبكة المحلية**: اكتشاف الأجهزة النشطة عبر فحص المنافذ (Socket Connect) لعنوان الـ IP.
3.  **تشفير الملفات**: نظام تشفير AES-256 حقيقي باستخدام مكتبة `cryptography`.
4.  **التعرف على الراوتر**: تحليل استجابات HTTP لتحديد العلامة التجارية (Huawei, TP-Link, ZTE).
5.  **اختبار السرعة**: قياس سرعة التحميل عبر طلبات HTTP حقيقية.
